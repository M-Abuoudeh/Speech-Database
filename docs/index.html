About the Project

This project aims to create a phonetic database for Jordanian dialects, ready for use in phonetic analysis and statistics. It provides a resource for researchers interested in studying phonetic similarities and differences across Jordanian dialects.

Collaborative Contribution

This project follows a **collaborative approach**. Researchers with access to the database are encouraged to contribute by submitting **analyses, improved transcriptions, or additional annotations**. To participate, please contact the project lead. This participatory approach not only ensures **data quality** but also fosters an **active scientific community** around the study of Jordanian Arabic dialects.

Financial Support

This project “Speech Database of Jordanian Arabic Dialects - SDJAD” is funded by Al-Hussein Bin Talal University with grant number 85/2022.

About the Researcher

This project was created and developed by **Mohammad Abuoudeh**, assistant professor at the Language and Linguistics Department in Al-Hussein bin Talal University, as the **Project Creator & Principal Researcher**. As a main contributor, I have been responsible for designing, collecting, and analyzing the data for the **Speech Database of Jordanian Arabic Dialects (SDJAD)**. I aim to provide a comprehensive and accessible phonetic database for researchers interested in Jordanian Arabic dialects.
