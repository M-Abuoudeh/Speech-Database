About the Project

The Speech Database of Jordanian Arabic Dialects (SDJAD) is a comprehensive phonetic resource aimed at documenting the linguistic landscape of Jordan. Data collection was conducted between **2023 and 2024**, gathering recordings from **109 speakers** (both male and female) representing diverse Jordanian governorates and origins.

Currently, the **Laylaa witheeb** task (Reading and Storytelling) has been fully processed, transcribed, and segmented. Additional tasks, including isolated words, semi-spontaneous conversation, and image description, are currently under processing and will be added to the database progressively.

Collaborative Contribution

This project follows a **collaborative approach**. Researchers with access to the database are encouraged to contribute by submitting **analyses, improved transcriptions, or additional annotations**. To participate, please contact the project lead. This participatory approach not only ensures **data quality** but also fosters an **active scientific community** around the study of Jordanian Arabic dialects.
